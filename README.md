# Example of README.md

Αυτό το αρχείο αποτελεί παράδειγμα README σε μορφή Markdown.
Στόχος του είναι να παρουσιάσει τη βασική δομή και τη χρήση τίτλων και παραγράφων.

## Σχετικά
Το Markdown είναι μια απλή γλώσσα μορφοποίησης που επιτρέπει τη δημιουργία δομημένου κειμένου χρησιμοποιώντας έναν απλό επεξεργαστή κειμένου.
